Основные методы:

cargoes_data_json - словарь, загруженный из файла json

filling_space_percent(cargoes_data_json)
  фактическая доля заполненного пространства
  
estimated_space_percent(cargoes_data_json)
  оцениваемая доля заполненного пространства

json_file_path - путь к файлу json

filling_space_percent_from_json_file(json_file_path)
  фактическая доля заполненного пространства

estimated_space_percent_from_json_file(json_file_path)
  оцениваемая доля заполненного пространства
